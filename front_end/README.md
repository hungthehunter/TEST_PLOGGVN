# 1. Công nghệ Sử dụng (Tech Stack)

+ Frontend Framework: React 19+

+ Ngôn ngữ: TypeScript

+ Bundler/Dev Server: Vite (sử dụng Rolldown)

+ Styling (Dựa trên front_end): Tailwind CSS, PostCSS, Autoprefixer

+ State Management (Nếu có): Nếu bạn sử dụng Redux, Zustand, hoặc Context API, hãy liệt kê ở đây.

+ Tối ưu hóa: babel-plugin-react-compiler

# 2. Cài đặt và Khởi chạy (Installation & Setup)

## A. Cài đặt các Dependency

### Clone repository
git clone [Đường dẫn Git của bạn]
cd onlinecv

### Cài đặt các gói
npm install
### HOẶC
yarn install

# 3. Cấu trúc thư mục (Folder Structure)

src/
├── components/   # Các thành phần UI tái sử dụng (Buttons, Cards, Icons)
├── pages/        # Các trang chính của ứng dụng (Home, Experience, Projects)
├── assets/       # Ảnh, phông chữ, video
├── styles/       # Tệp CSS/Tailwind CSS chính
├── types/        # Các định nghĩa kiểu TypeScript (interfaces, types)
└── main.tsx      # Điểm khởi đầu của ứng dụng
